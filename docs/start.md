## 特性

- `import` 别名
- 支持自定义vite配置
- <SvgIcon name="gongju" /> 组件自动注册
- <SvgIcon name="zujian" /> 集成 UI 库（element-plus）
- <SvgIcon name="tailwindcss" /> 集成 tailwindcss
- <SvgIcon name="svg" class="text-lg" /> SVG图标（自动拉取iconfont）
